# Fabric

Fabric is used for Minecraft mod development.

## Installation

Install [Deno](https://deno.com/) for the [Fabric CLI](https://fabricmc.net/develop/cli/).

## Usage

### Initialization

```ps
fabric init <name>
```

Use Kotlin. Package name is `com.<username>`.

Run `vscode` and `genSources` Gradle task using VS Code Gradle Extension, or:

```ps1
.\gradlew vscode
.\gradlew genSources
```
