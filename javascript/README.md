# JavaScript

JavaScript is used for frontend web development.

## Installation

Install [Deno](https://deno.com/). For CSS Preprocessor, install [Sass](https://sass-lang.com/).

## Usage

### Initialization

#### Dependencies

```sh
deno add <dep1> <dep2>
deno add <dep3> <dep4> -D # dev dependencies
```

[Dependencies](deps.json)

## Docs

```sh
deno doc
```

## Upgrading

After upgrading, run:

```sh
deno clean
```
